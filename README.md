# Spirit Flower Clock

An animated Flower of Life clock built with React + Vite.

## Distributable build (for non-developers)

To produce a downloadable package:

```bash
npm install
npm run build:dist
```

This creates:

- `dist/` — static web app output
- `flower-clock-dist.zip` — ready-to-share distributable archive

Anyone can unzip `flower-clock-dist.zip` and open `index.html` directly in a browser (no Node.js or dev tooling required).

## Developer commands

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
